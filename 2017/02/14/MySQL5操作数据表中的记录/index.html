MySQL(五)--操作数据表中的记录

发表于 2017-02-14 | 分类于 MySQL | | 阅读量

之前是对字段的操作，现在呢来学习一下对记录的操作~

MySQL插入记录INSERT

  • col_name省略时，默认要插入所有的字段的内容，其中，AUTO_INCREMENT字段，不能不赋值，要想让其按照自增的语句默认赋值，在插入时，在相应位置插入DEFAULT或NULL即可。具有DEFAULT约束的字段，将其设为默认值，在相应位置插入DEFAULT或设置的值即可。
  • 可以同时插入多条记录，每一个小括号里是一条记录，不同的字段用逗号隔开，括号间用逗号个隔开
  • 插入的记录可以为运算符，可以为表达式

MySQL插入记录INSERT SET-SELECT

INSERT SET方法

  • 与第一种方式的区别是可以使用子查询
  • 不能同时插入多条语句
    例如：
  • 想插入哪个，是以表达式方式赋值的

INSERT SELECT方法

  • 此方法可以将查询结果插入到指定数据表

MySQL单表记录更新UPDATE

  • UPDATE名令到底是干嘛的呢，记录书写的过程中，想更改一下字段值。
  • 省略条件将更新所有的字段记录

MySQL单表删除记录DELETE

  • 当删除某些记录时，表中的自增的字段比如id， 会为目前最大的id+1 ，而不是去补充原来删除的id号

MySQL查询表达式解析

  • 数据表中的SELECT语句的使用率啊基本上占所有语句使用的80%以上
  • 查询表达式，每一个表达式表示想要的一列，必须有至少一个
  • 多个列之间以英文逗号分离
  • 星号()表示所有列。`tbl_name.`可以表示命名表的所有列
  • 查询表达式可以使用[AS] alias_name为其赋予别名
  • 别名可用于GROUP BY,ORDER BY或HAVING子句

MySQL where语句进行条件查询

  • 对记录进行过滤，如果没有指定WHERE子句，则显示所有记录
  • 在WHERE表达式中，可以使用MySQL支持的函数或运算符

having语句设置分组条件

  • HAVING语句要么为一个聚合函数，聚合函数就是AVG(),SUM()这种函数，永远只有一个返回结果。要么这个字段出现在前面的SELECT语句当中
