MySQL(零)--MySQL的安装与配置

发表于 2017-02-10 | 分类于 MySQL | | 阅读量

安装环节

笔者安装的是目前最新的5.7版本。
首先，要去MySQL官网下载MySQL服务器安装软件包，我下载为版本mysql-installer-community-5.7.16.0.msi

  1. 在设置界面，由于之前没有相关开发经验，所以选择默认的Default版本Developer Default
  2. 进入安装前环境验证，它会自动检测你电脑上缺少的包，一些包会为你自动安装，一些包标有（manual）需要手动安装。点击“Execute”，就会一个个地为我们自动安装包，需要手动安装的，就要去官网下载安装包，安装下来就好，点击”check“，会检测这个包是否安装成功
  3. 之后都选择默认配置，一路“next”下去
  4. 到添加用户，设置密码的环节，就为mysql设置密码
  5. 然后再一路“next”下去就可以了

测试环节

怎么知道自己的MySQL是安装成功了没有呢？

输入show databases；
当当当当~~有如下的输出就说明我们安装成功啦

方法二：用cmd命令行

  1. 将MySQL的路径加到环境变量Path中。
    一般来说，MySQL的默认安装路径在C:\Program Files\MySQL\MySQL Server 5.7\bin，Path是右键计算机->属性->高级->环境变量，在系统变量里面找到Path，将MySQL的路径加到里面
  2. 在DOS命令窗口输入 mysql -h localhost -u root -p回车 进入mysql数据库
    其中-h表示服务器名，localhost表示本地，当然可以写成127.0.0.1
    -u为数据库用户名，root是mysql默认用户名；
    -p为密码，如果设置了密码，可直接在-p后链接输入，如：-p123456，用户没有设置密码，显示Enter password时，直接回车即可。

MySQL的目录结构

  • bin目录：存储可执行文件
  • data目录：存储数据文件
  • docs：文档
  • include目录：存储包含的头文件
  • lib目录：存储库文件
  • share目录：错误消息和字符集文件
  • my.ini文件为配置文件（我的my.ini是在C:\ProgramData\MySQL\MySQL Server 5.7目录下的）

MySQL的启动与停止

  • 在操作系统层面，可以在服务处启动或停止MySQL，我的MySQL服务就叫MySQL57，大家可以来这看一下自己的版本服务名字叫什么
  • 在命令行层面：cmd进入命令行，net start mysql57可以启动MySQL服务；net stop mysql57可以停止MySQL服务
    如果发生系统错误5，拒绝访问，则用管理员权限运行cmd即可，具体的操作为右键cmd->以管理员身份运行。
