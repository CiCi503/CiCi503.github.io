Tomcat安装及配置

发表于 2017-01-23 | 分类于 IDEA | | 阅读量

Tomcat安装

  1. 在Apache官网下载相应的压缩包。
  2. 是一个绿色软件，解压缩到需要安装的文件夹(博主是解压缩到了C盘)
  3. 新建环境变量CATALINA_HOME，位置是在安装包解压缩后的位置。
  4. 安装包下有一个bin目录，里面有startup.bat，windows系统运行该文件，Unix/Mac OS/Linux运行startup.sh文件
  5. 在浏览器中输入http://localhost:8080/查看
  6. 终止时，运行bin目录下的shutdown.bat或shutdown.sh

Tomcat目录结构

  • /bin目录：存放各种平台下用于启动和停止Tomcat的命令文件
  • /conf目录：存放Tomcat服务器的各种配置文件
  • /lib目录:存放Tomcat服务器所需的各种JAR文件
  • /logs：存放Tomcat的日志文件
  • /temp目录：Tomcat运行时用于存放临时文件
  • /webapps目录：当发布Web应用时，会将Web应用的文件发布到此目录中
  • /work目录：Tomcat把由JSP生成的Servlet放于此目录下

手工编写第一个Web程序

  1. 在WebApps创建项目目录
    • 新建一个文件夹，里面有index.jsp文件和Web-INF目录
  2. 编写index.jsp（内容为html格式的内容）
  3. 创建Web-INF目录
    • 里面有一个classes目录，空目录即可
    • 一个lib目录，空目录即可
    • 一个web.xml文件，文件的格式可以去WebApps里面的examples里面复制
  4. 测试运行
    • 打开http://localhost:8080，继续输入myhome/index.jsp，即http://localhost:8080/myhome/index.jsp
    • 就可以看到index.jsp里面的html页面了

Web-INF目录结构

  • Web-INF目录是Java的Web应用的安全目录，所谓安全目录就是客户端无法访问，只有服务端可以访问的目录
  • web.xml项目部署文件（可以用<welcome-file-list>标签来修改欢迎界面，默认的欢迎界面是index.jsp文件的内容）
  • classes文件夹，用于防止*.class文件
  • lib文件夹，用于放置需要的jar包
