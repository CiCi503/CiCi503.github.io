JSP简介

JSP全名为Java Server Pages，根本是一个简化的Servlet设计，他实现了在Java当中使用HTML标签。JSP是一种动态网页技术标准，也是JAVAEE的标准。JSP与Servlet一样,是在服务器端执行的。

常见的动态网站开发对比

  • JSP：Java平台，安全性高，适合开发大型的，企业级的Web应用程序。
  • Asp.net：简单易学，但是安全性及跨平台性差。
  • PHP：简单，高效，成本低开发周期短，适合中小企业的Web应用开发（LAMP:Linux+Apache+MySQL+PHP)

Cookie到底是什么呢？听起来很好吃的样子~

http协议无状态性

无状态是指，当浏览器发送请求给服务器的时候，服务器响应客户端请求。
但是当同一个浏览器再次发送请求给服务器的时候，服务器并不知道它就是刚才那个浏览器
简单地说，就是服务器不会去记得你，所以就是无状态协议。

Cookie概述

因为Http是无状态的，所以要想办法保存用户状态

  • Session
  • Cookie(客户端)

Tomcat安装及配置

发表于 2017-01-23 | 分类于 IDEA | | 阅读量

Tomcat安装

  1. 在Apache官网下载相应的压缩包。
  2. 是一个绿色软件，解压缩到需要安装的文件夹(博主是解压缩到了C盘)
  3. 新建环境变量CATALINA_HOME，位置是在安装包解压缩后的位置。
  4. 安装包下有一个bin目录，里面有startup.bat，windows系统运行该文件，Unix/Mac OS/Linux运行startup.sh文件
  5. 在浏览器中输入http://localhost:8080/查看
  6. 终止时，运行bin目录下的shutdown.bat或shutdown.sh

Eclipse快捷键

发表于 2016-07-22 | | 阅读量
  • 快速修正：Ctrl+1
  • 单词补全：Alt+/
  • 打开类型：Ctrl+Shift+T 即打开一个类型，显示源码
  • 删除行：Ctrl+D
  • 在当前行上插入一行：Ctrl+Shift+Enter
  • 在当前行下插入一行： Shift+Enter
  • 上下移动选中的行：Alt+Up/Down
  • 上一个/下一个光标的位置：Alt+Left/Right
  • 上一个/下一个成员（成员对象或成员函数）：Ctrl+Shift+Up/Down
  • 跳到某行：Ctrl+L
  • 行末/行首：End/Home
  • 选中到行末/行首：Shift+End/Home
  • 前一个/后一个单词：Ctrl+Right/Left
  • 选中上一个/下一个单词：Ctrl+Shift+Left/Right
  • 目当前项运行，相当于那个绿色小按钮： Ctrl+F11
